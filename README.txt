Mali-G77-MC9 Reverse Engineering
=================================

Reverse engineering project for the ARM Mali-G77 MC9 GPU.

## Project Structure

- `src/kbase/` - Kernel driver analysis and test programs
- `scripts/` - Debug and testing scripts
- `findings/` - Documentation of reverse engineering discoveries

## Quick Start

See AGENTS.md for detailed instructions on:
- Connecting to the device via SSH/Termux
- Running test programs
- Debugging GPU operations

## Requirements

- Termux on Android device with Mali-G77 GPU
- Root access on device
- SSH access (port 8022)
